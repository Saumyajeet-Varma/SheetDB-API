# SheetDB API

## Problem

Millions of non-tech users rely on Excel to manage structured data such as inventory, student records, HR data, sales tracking, customer information, etc.
How ever, Excel is not a database.

There is a gap between Excel familiarity and database power.

## Solution

**SheetDB**
A System that allows users to use excel as they normally do while all data operations are excuted on a real database through a secure API layer.

> Excel becomes a smart frontend for structured database.
